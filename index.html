使い方

閲覧（ログイン不要）

  • 公開トリップ: 右側パネルからトリップを選択して閲覧
  • 写真: サムネイルクリックで拡大、地図マーカー表示
  • 操作: 「▶ 再生」「前へ」「次へ」でスライドショー
  • 間隔: 1秒/3秒/5秒で切り替え可能

編集（ログイン必要）

  • ログイン: メニュー（≡）→「ログイン」で編集モードを有効化
  • 写真アップロード: メニュー内のアップロードゾーンにドラッグ＆ドロップ（GPS付き写真を推奨）
  • GPX読み込み: ルートファイルをアップロードで地図に重ね表示
  • トリップ保存: トリップ名・説明・URLを入力し「保存」
  • 公開設定: 「公開する」にチェックでログインなし閲覧可能に
  • 写真の詳細: 再生中に✎ボタンでランドマーク番号・説明・URLを編集

公開トリップのデプロイ

  • トリップを「公開する」で保存
  • 「📤 公開トリップをエクスポート」で public-trips.json をダウンロード
  • airgo フォルダに配置して Cloud Run にデプロイ

設定（ログイン時）

  • AI API: OpenAI または Google Gemini を選択
  • API Key: 選択したサービスのAPI Keyを入力して保存（ブラウザ内に保存）

※ GPS情報が必要です。スマートフォンで「位置情報をオン」にして撮影した写真をご利用ください。

公開トリップの表示設定

セクションでトリップをグループ化し、表示順を変更できます。右側パネルの↑↓ボタンでも並べ替え可能です。

トリップの表示/非表示

チェックを外すと右側の公開トリップ一覧に表示されません。
